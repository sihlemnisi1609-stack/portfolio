# Personal Portfolio Website
A professional landing page designed to showcase my academic background and software development projects.

## Project Overview
This project serves as a central hub for my professional identity. It highlights my skills, provides a direct link to my GitHub repositories, and hosts a digital version of my CV.

## Technologies Used
* **HTML5**: For structure and content.
* **CSS3**: For styling and layout.

## Features
* **Bio Section**: Brief introduction to my background.
* **Project Links**: Direct access to my coding work.
* **Contact Information**: Easy way to reach me for professional opportunities.
